如何查询电脑配置？

Windows系统查询方法

1 查看操作系统版本

方法1：按下 Win + R，输入 winver 并回车，在弹出窗口中查看版本信息
方法2：打开"设置" → "系统" → "关于"，在"Windows规格"中查看版本号

2 查看处理器类型

打开"设置" → "系统" → "关于"，在"设备规格"的"处理器"一栏查看（通常显示为Intel或AMD开头）

3 查看内存大小

打开"设置" → "系统" → "关于"，在"设备规格"的"已安装内存(RAM)"一栏查看

macOS系统查询方法

1 查看操作系统版本

点击屏幕左上角苹果图标 → "关于本机"，在弹出窗口中查看"macOS版本"

2 查看芯片类型

点击屏幕左上角苹果图标 → "关于本机"，在"处理器"或"芯片"一栏查看（显示为Intel或Apple M1/M2/M3/M4）

3 查看内存大小

点击屏幕左上角苹果图标 → "关于本机"，在"内存"一栏查看
